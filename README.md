> 이는 깃허브 등 인터넷에서 다운로드 받은 앱에 대해 맥(macOS)이 적용하는 기본 보안 조치입니다. 바이러스나 파일 손상이 아니니 안심하세요! 아래 단계를 따라 딱 한 번만 허용해주시면 다음부터는 정상 실행됩니다.
> 
> 1. 다운로드 받으신 **`Ggeolmu Language Pro.app`** 앱 아이콘을 마우스 **우클릭(또는 두 손가락 클릭)** 합니다.
> 2. 메뉴 모음에서 맨 위에 있는 **[열기]**를 클릭합니다.
> 3. 경고창이 뜨면 당황하지 마시고, 다시 한 번 **[열기]** 버튼을 클릭합니다.
> 4. (끝!) 이제부터는 그냥 더블클릭해도 정상적으로 열립니다.

Ggeolmu Language Pro는 유튜브 영상을 보며 모르는 단어를 바로바로 찾아볼 수 있는 로컬 어학 학습 프로그램입니다.

## 🚀 주요 기능
- **자동 자막 추출**: 유튜브 주소만 넣으면 스크립트를 깔끔하게 뽑아줍니다.
- **마우스 오버 번역**: 모르는 단어에 마우스를 올리면 뜻과 발음을 바로 알려줍니다.
- **클릭 이동**: 스크립트의 단어를 누르면 영상이 해당 위치로 이동하여 재생됩니다.
- **즐겨찾기 버튼**: 유튜브 창에서 즐겨찾기 버튼만 누르면 바로 학습 창으로 화면이 넘어갑니다.
- **스페이스바 재생/정지**: 키보드 스페이스바만으로 영상을 쉽게 제어할 수 있습니다.

## ⌨️ 키보드 단축키 (Shortcuts)
편리한 학습을 위해 키보드 조작법을 제공합니다:
- **`Space`** : 영상 재생 / 일시정지
- **`v`** : 이전 문장(블럭)으로 이동하여 처음부터 재생
- **`b`** : 현재 읽고 있는 문장(블럭) 처음부터 다시 듣기
- **`n`** : 다음 문장(블럭)으로 건너뛰어 재생
- **`g`** : 현재 재생 중인 단어의 **앞 단어**로 되감기
- **`1` ~ `4`** : 각각 영상 전체 길이의 **0%, 25%, 50%, 75%** 위치로 이동

사용자가 클릭할 수 있는 **컨트롤 패널 앱 1가지**와 **핵심 파일들**로 이루어져 있습니다.

- 🎛️ `Ggeolmu Language Pro.app` : 프로그램을 켜고 끄는 통합 컨트롤 패널입니다. (더블클릭해서 사용)
- 📂 `LanguageLearner/` : 프로그램의 디자인과 기능이 들어있는 핵심 폴더입니다. (수정할 필요 없음)

## ⚙️ 실행 가이드 (어떻게 켜나요?)

기존의 복잡한 여러 앱 실행 방식 대신, 이제 **단 하나의 컨트롤 패널 앱**으로 통합되었습니다!

1. 깃허브에서 프로젝트를 다운로드한 후, 폴더 안의 **`Ggeolmu Language Pro.app`** 앱을 더블클릭하여 실행합니다.
2. 나타나는 창에서 **▶ 실행 (Start)** 버튼을 클릭합니다.
   * 첫 실행 시 로컬 서버 설정을 위해 **맥북 로그인 비밀번호**를 한 번 요구합니다.
3. "서버 실행 완료!" 안내 창이 나타나면 **[접속하기]** 버튼을 클릭합니다.
   * 크롬 브라우저가 자동으로 열리며 학습 프로그램(`http://ggeolmu-language.com`)이 시작됩니다.

### 서버 종료 방법
1. 다시 **`Ggeolmu Language Pro.app`** 앱을 더블클릭하여 실행합니다.
2. **■ 종료 (Stop)** 버튼을 클릭합니다.
   * "종료 완료" 메시지가 뜨며 모든 네트워크 설정이 깔끔하게 원상복구됩니다.
