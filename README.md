# Ggeolmu Language Pro

Ggeolmu Language Pro는 유튜브 영상을 보며 모르는 단어를 바로바로 찾아볼 수 있는 로컬 어학 학습 프로그램입니다.

---

## 📦 다운로드 및 시작 가이드

### 1단계: 다운로드 (Download)
1. 현재 깃허브 페이지 우측 상단에 있는 초록색 **[<> Code]** 버튼을 클릭합니다.
2. 메뉴에서 **[Download ZIP]**을 선택하여 파일을 다운로드 받습니다.
3. 다운로드 폴더에서 `EngPractice-main.zip` 파일의 압축을 풉니다.

### 2단계: 첫 실행 및 보안 알림 해결 (중요!)
압축을 푼 폴더 안에 있는 **`Ggeolmu Language Pro.app`**을 처음 실행할 때, 맥(macOS)의 보안 정책으로 인해 "확인되지 않은 개발자가 배포했기 때문에 열 수 없습니다"라는 경고창이 뜹니다. 당황하지 마시고 **딱 한 번만** 아래 단계를 거쳐 허용해 주세요!

> [!WARNING]
> **⚠️ 첫 실행 시 막혔을 때 허용하는 방법**
> 1. 다운로드 받으신 **`Ggeolmu Language Pro.app`** 앱을 우클릭 후 **[열기]**를 누릅니다.
> 2. 열기 버튼이 없고 실행이 아예 차단된다면, 창을 닫고 맥북 화면 좌측 상단의 **(애플 로고)**를 눌러 **[시스템 설정...]**에 들어갑니다.
> 3. 좌측 메뉴에서 **[개인정보 보호 및 보안]**을 클릭하고, 마우스를 조금 아래로 내립니다.
> 4. "보안" 섹션에 보면 *"Ggeolmu Language Pro이(가) 확인되지 않은 개발자..."* 라는 문구 옆에 **[확인 없이 열기]** 버튼이 있습니다. 이를 클릭합니다.
> 5. 나타나는 팝업에서 **[열기]**를 클릭하시면 성공적으로 앱이 열립니다! (이후부터는 편하게 더블클릭하시면 됩니다)

### 3단계: 서버 실행 및 프로그램 접속
기존의 복잡한 실행 방식 대신, 이제 **단 하나의 컨트롤 패널 앱**으로 모든 것을 조작합니다!

1. 앱이 켜지면 나타나는 패널에서 **▶ 실행 (Start)** 버튼을 클릭합니다.
   * *(첫 실행 시 내부 네트워크 설정을 위해 맥북 로그인 비밀번호를 1회 요구합니다.)*
2. "서버 실행 완료!" 안내 창이 나타나면 **[접속하기]** 버튼을 클릭합니다.
   * 크롬 브라우저가 자동으로 열리며 학습 프로그램(`http://ggeolmu-language.com`)이 시작됩니다.

---

### 🛑 서버 종료 방법
1. 프로그램을 다 쓰셨다면 다시 **`Ggeolmu Language Pro.app`** 앱을 엽니다.
2. **■ 종료 (Stop)** 버튼을 클릭합니다.
   * "종료 완료" 메시지가 뜨며 켜져있던 보이지 않는 서버들이 깔끔하게 꺼집니다.
