# Ggeolmu Language Pro

Ggeolmu Language Pro는 유튜브 영상을 보며 모르는 단어를 바로바로 찾아볼 수 있는 로컬 어학 학습 프로그램입니다.

## 🚀 주요 기능
- **자동 자막 추출**: 유튜브 주소만 넣으면 스크립트를 깔끔하게 뽑아줍니다.
- **마우스 오버 번역**: 모르는 단어에 마우스를 올리면 뜻과 발음을 바로 알려줍니다.
- **클릭 이동**: 스크립트의 단어를 누르면 영상이 해당 위치로 이동하여 재생됩니다.
- **즐겨찾기 버튼**: 유튜브 창에서 즐겨찾기 버튼만 누르면 바로 학습 창으로 화면이 넘어갑니다.
- **스페이스바 재생/정지**: 키보드 스페이스바만으로 영상을 쉽게 제어할 수 있습니다.

## ⌨️ 키보드 단축키 (Shortcuts)
편리한 학습을 위해 키보드 조작법을 제공합니다:
- **`Space`** : 영상 재생 / 일시정지
- **`v`** : 이전 문장(블럭)으로 이동하여 처음부터 재생
- **`b`** : 현재 읽고 있는 문장(블럭) 처음부터 다시 듣기
- **`n`** : 다음 문장(블럭)으로 건너뛰어 재생
- **`g`** : 현재 재생 중인 단어의 **앞 단어**로 되감기
- **`1` ~ `4`** : 각각 영상 전체 길이의 **0%, 25%, 50%, 75%** 위치로 이동

## 📁 프로그램 파일 구조 (Structure)

사용자가 클릭할 수 있는 **실행 앱 4가지**와 **핵심 폴더 1가지**로 이루어져 있습니다.

- 🟢 `Set_Local_DNS.app` : 나만의 접속 주소(`ggeolmu-language.com`)를 컴퓨터에 등록해 주는 앱 (최초 1회만 실행)
- 🔴 `Unset_Local_DNS.app` : 등록했던 주소를 다시 지우고 싶을 때 사용하는 앱
- 🚀 `Start_Server.app` : 학습 프로그램을 켜는 앱 (사용할 때마다 실행)
- 🛑 `Stop_Server.app` : 학습 프로그램을 완전히 끄는 앱
- 📂 `LanguageLearner/` : 프로그램의 디자인과 기능이 들어있는 핵심 폴더 (수정할 필요 없음)

## ⚙️ 실행 가이드 (어떻게 켜나요?)

프로그램을 처음 사용하시거나 켤 때는 딱 **2단계**만 기억하시면 됩니다.

**[최초 1회 세팅]**
1. **`Set_Local_DNS.app`** 을 더블클릭해서 실행하세요. 
   - 비밀번호를 입력하면 컴퓨터에 `ggeolmu-language.com` 주소가 등록됩니다.

**[평소에 사용할 때]**
2. **`Start_Server.app`** 을 더블클릭해서 실행하세요.
   - 비밀번호를 입력하면 뒤에서 서버가 조용히 켜집니다.
   - 브라우저를 열고 주소창에 `http://ggeolmu-language.com` 을 입력하고 들어가서 공부를 시작하세요!

**[다 쓰고 끌 때]**
- 다 쓰셨다면 **`Stop_Server.app`** 을 더블클릭해서 프로그램을 완전히 꺼주시면 됩니다.
